## Tasks

Complete the following tasks in your Jupyter Notebook:

1. **Define total_sales Function** - Create a function `total_sales` that accepts one parameter
2. **Add Second Parameter** - Update the function to accept both `price` and `num_tickets` parameters
3. **Calculate Total Sales** - Implement logic to multiply price and num_tickets
4. **Round to 2 Decimals** - Use the round() function to limit output to two decimal places

## Function Examples
```python
# Example usage:
total_sales(15.99, 1001)  # Should return 16005.99
total_sales(16.99, 1232)  # Should return 20931.68
```
